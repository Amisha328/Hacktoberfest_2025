What is Hacktoberfest?

Hacktoberfest is an annual celebration of open-source software. Contributors submit high-quality pull requests to participating repositories, learn, and grow their skills. Maintainers welcome new contributors and improve project health.

  • Quality over quantity: Aim for thoughtful, documented changes.
  • Be respectful: Read the repository README, CONTRIBUTING, and CODE_OF_CONDUCT.
  • Communicate: Open issues, ask clarifying questions, and follow review feedback.

How to Participate

1. Pick a project

Look for repos with clear documentation and active maintainers. Favor topics like good-first-issue and help-wanted.

2. Read the docs

Understand setup, coding standards, and branching strategy. Confirm whether the repo accepts Hacktoberfest PRs.

3. Communicate

Comment on an issue to get assigned, or open a new one describing your plan before coding.

4. Make focused PRs

Keep changes small and testable. Explain the "why" and "how" in your PR description with before/after details.

Maintainer Tips

  • Label well: Add good-first-issue and help-wanted with clear scope.
  • Provide templates: Issue and PR templates reduce back-and-forth.
  • Be responsive: Acknowledge newcomers quickly with guidance and kindness.
  • Automate checks: Linting and tests keep quality high and reviews fast.
